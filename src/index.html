Proyectos

-Game Informer-

Desarrollo de una Spa (Simple Page Application) utilizando React y Redux como state management, Todos los componentes fueron desarrollados con CSS sin uso de librerías externas a modo de práctica. La SPA consume datos de una API (RAWG.io) a través de un Back End desarrollado en Node.JS utilizando Express. Dicha aplicación cuenta con un listado de 705 videojuegos mostrando información detallada (imágenes, géneros, fecha de lanzamiento, etc.)Si desea se puede filtrar y/u ordenar, crear, eliminar y guardar como favoritos.

-EasyLearning-

Desarrollo de una plataforma e-Learning de forma remota entre 7 colaboradores de Argentina, implementando scrum como metodología ágil y Git/GitHub como control de versiones. Se trata de una colección de cursos que puede crear cualquier usuario. La aplicación cuenta con una autenticación con Firebase, un dashboard donde se puede controlar toda la aplicación siendo admin(eliminar usuarios, aprobar/eliminar cursos, ver estadísticas, etc.) cuenta con una pasarela de pago para los usuarios que deseen adquirir un curso, carrito persistente, reviews de la página, curso y videos. Creación de usuarios, modificación de usuarios, creación de cursos, modificación de cursos, upload de imágenes con Cloudinary.
